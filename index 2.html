About AccessiBuild

AccessiBuild is a community-driven platform that helps people discover, rate, and review venues based on their accessibility features. Our mission is to make every space welcoming and accessible to everyone.

Functional Requirements

FR1Venue Discovery & Search

  • Users can search venues by name, category, or location
  • Categories include Café, Library, Entertainment, Gym, Museum, and Restaurant
  • Results update in real-time as users type

FR2Accessibility Filtering

  • Three accessibility tags: Mobility, Sensory, and Visual
  • Users can filter by multiple tags simultaneously
  • Filters combine with search and category for precise results

FR3Community Ratings & Reviews

  • 1–5 star accessibility rating system
  • Detailed text reviews with accessibility-specific tags
  • Review dates and user attribution

FR4Multimedia Evidence

  • Venue images showing accessibility features
  • Photo upload capability for community contributions

FR5Geolocation

  • Venue coordinates stored for mapping
  • Customisable radius search around user location

Technology

This standalone version is built with pure HTML, CSS, and JavaScript — no frameworks or build tools required. Simply open the file in any modern browser.
